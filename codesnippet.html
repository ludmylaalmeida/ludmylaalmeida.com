Project Overview.

The goal of the project was to develop a web application where students can find a collection of helpful tools necessary to learn a new programming language. The website will provide definitions of computer science concepts, along with well written and well-thought code in the three most prominent programming languages: C, C++, and JAVA. It will also include code "snippet" with a re-usable source code.

This tool will be a reference for students who are learning a new language since they can compare how to write code for the same concept in a different programming language. The website will be essential for people who want to develop the skills necessary to succeed in the tech industry.
